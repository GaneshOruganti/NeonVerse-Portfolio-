# **App Name**: NeonVerse Portfolio

## Core Features:

- 3D Animated Hero Section: Interactive landing page with a rotating 3D Earth or neural network background, animated intro text, and floating social icons. React Three Fiber.
- Holographic Photo Display: Showcase a professional photo within a rotating holographic frame in the 'About Me' section. Animated timeline to highlight key experiences.
- Interactive Skill Map: Animate and group the technologies in an interactive skill map with glow pulse effects on hover.
- 3D Project Showcase: Showcase the project cards as rotating 3D elements which open into modals when selected. The modals contain project descriptions, technologies, GitHub and live demo links. Lazy loading for performance.
- AI-Powered Personalized Recommendations Showcase: Integrate a showcase featuring the Personalized Product Recommendation System using LLMs. Include a brief explanation of how the tool is implemented.
- Animated Stepping Stone Timeline: Present experiences in leadership and roles chronologically, using animated stepping stones.
- Clean Contact Section: Design an elegant and straightforward contact section with floating form fields, direct links to email, LinkedIn, and GitHub, with a glowing CTA.

## Style Guidelines:

- Primary color: Electric indigo (#3B82F6) for a futuristic and premium feel.
- Background color: Dark slate gray (#050816) - a heavily desaturated tone to support the dark theme, yet distinct from pure black.
- Surface color: (#101B33)
- Accent color: Sky blue (#00F5FF) - an analogous color providing high contrast for glowing highlights and calls to action.
- Text color: (#F1F5F9)
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern and neutral aesthetic.
- Use neon-glowing icons with a minimalist design style.
- Floating UI elements with soft glassmorphism to enhance the futuristic look.
- Smooth scrolling transitions and subtle animations using React Three Fiber and GSAP to provide a premium user experience.
